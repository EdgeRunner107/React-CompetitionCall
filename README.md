# React-CompetitionCall
리액트에서 공모전 목록크롤링


기존 프로젝트 형식에 편승한 형태  (정확한 리액트 프로젝트아님)


백엔드 - NODE.JS과 프론트 -React 와의 통신을 테스트

시사점
NODE에서 크롤링 데이터를 SQL 에 저장함 (중복제거)
React에서 node api에 요청하여 (gpt가 인정함)  데이터를 목록화
